# 👾 Alien Shooter

A simple **2D Alien Shooter** game built using **Python** and **Pygame**.

## 🎮 Features

* Alien ships spawn continuously with increasing difficulty.
* Shoot aliens to earn points.
* Sound effects and background music.
* Game Over screen with restart functionality (`R` to play again).

## 🛠️ Tech Stack

* Python
* Pygame

## 🚀 How to Run

```bash
pip install pygame
python main.py
```

## 🎯 Controls

| Key     | Action                    |
| ------- | ------------------------- |
| ↑ ↓ ← → | Move Player               |
| Space   | Shoot                     |
| R       | Restart (after Game Over) |

## 📁 Project Structure

This project was built as a fun way to practice **game development with Pygame**, including game loops, collision detection, object management, and sound integration.
